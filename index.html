    • libusb

      • mkdir libusb-legacy
      • tar --strip-components=1 --directory=libusb-legacy -xzf libusb-legacy.tar.gz
      • cd libusb-legacy
      • CPPFLAGS=-m32 LDFLAGS=-m32 ./configure
      • make
      • sudo make install
